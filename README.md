# Vansh Pratap Gautam — Portfolio

Personal portfolio website built with **Next.js 16**, **Tailwind CSS 4**, and **Motion**.

## Features

- ⚡ Next.js 16 with App Router
- 🎨 Tailwind CSS v4 with dark/light mode
- 🖱️ Interactive splash cursor (WebGL)
- 📜 Scroll-based parallax project tiles
- 🎭 Smooth page transitions with Motion
- 📱 Fully responsive design
- 🔍 SEO optimized with meta tags & sitemap


## Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS v4
- **Animations**: Motion (Framer Motion) + GSAP
- **Smooth Scroll**: Lenis
- **Deployment**: Vercel
